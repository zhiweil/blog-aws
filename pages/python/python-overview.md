## Installation & Execution
```bash
# for fedora and RHEL
suod dnf install -y python3

# run it
python3

```

To quit from python CLI, simple run quit(), exit() or CTRL+Z.

## Comments
* one line comment: a line starts a "#" sign.
* multi-line comment: use ''' ... ''' or """ ... """. Be careful with the indentation.
